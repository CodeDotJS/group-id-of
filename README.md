## Install

```
$ npm install --global group-id-of
```
__OR__

```
$ sudo npm install --global group-id-of
```
## Usage

```
$ group-id-of

	Usage   : group-id-of <group's name>

	Example : group-id-of linuxscreenshots

	Help    : group-id-of -h --help
```
